# 麦小吉 (Mai Xiaoji) - 中国农业大学 AI 智能体网页版

这是一个基于 React 的网页应用，专门用于承载中国农业大学新生助手“麦小吉”的 Coze 智能体。

## 🚨 为什么打开是空白的？(必读)

**不要直接双击 `index.html` 打开！**

因为这个网页使用了专业的 `React` 和 `TypeScript` 技术编写，浏览器本身看不懂这些高级代码。你需要一个“翻译器”（开发服务器）来运行它。

请按照下方的 **【超简单启动教程】** 操作，只需 3 步即可看到画面。

---

## 💻 超简单启动教程 (本地运行)

### 第一步：下载并安装环境
1. **下载代码**：将本项目的所有文件保存在一个文件夹里（比如叫 `cau-xiaoji`）。
2. **安装 Node.js**：去官网 [nodejs.org](https://nodejs.org/) 下载安装包（推荐下载 LTS 版本），一路点击“下一步”安装。
   - *如何验证安装成功？* 按 `Win + R`，输入 `cmd` 回车，输入 `node -v`，如果有版本号跳出来就对了。

### 第二步：安装依赖
1. 在你的项目文件夹（`cau-xiaoji`）里，在空白处 **按住 Shift 键 + 点击鼠标右键**，选择 **“在此处打开 Powershell 窗口”** 或 **“在终端打开”**。
2. 在黑底白字的窗口里输入以下命令，然后按回车：
   ```bash
   npm install
   ```
   *(这时候屏幕会滚动很多代码，是在自动下载需要的工具包，等待它停下来)*

### 第三步：启动网页
1. 依赖安装完成后，继续输入：
   ```bash
   npm run dev
   ```
2. 当你看到类似 `Local: http://localhost:5173/` 的绿色字样时，说明成功了！
3. **按住 Ctrl 键** 并点击那个网址（或者复制网址到浏览器打开），你就看到麦小吉的网页了！

## 🌐 进阶：如何发布给同学用？(不用自己电脑挂机)

如果你想把网页发给全校同学用，推荐使用 **Vercel** 免费托管：

1. 注册一个 [GitHub](https://github.com/) 账号。
2. 下载 [GitHub Desktop](https://desktop.github.com/) 软件，把你的项目文件夹上传到 GitHub。
3. 注册 [Vercel](https://vercel.com/)，用 GitHub 账号登录。
4. 点击 "Add New Project"，选择你刚才上传的项目。
5. 点击 "Deploy"（部署）。
6. 等待 1 分钟，Vercel 会给你一个网址（例如 `mai-xiaoji.vercel.app`），这个网址就可以发到朋友圈和新生群里啦！

## 🤖 Coze 智能体配置指南

### 1. 身份与逻辑设定 (System Prompt)

请复制以下内容到 Coze 编排页面的 **人设与回复逻辑** 区域：

```markdown
# Role
你叫“麦小吉”，是中国农业大学的专属AI新生助手。你的形象是热情、专业、耐心的学长/学姐。
你的核心任务是根据用户的身份（本科新生 vs 研究生新生）提供差异化的服务。

# Workflow (工作流)
1. **身份确认**：
   - 在对话开始时，如果用户未表明身份，先礼貌询问：“嗨！我是麦小吉。请问你是本科新同学还是研究生新同学呢？不同阶段我有不同的秘籍传授哦！”

2. **分支服务逻辑**：
   - **Scenario A: 用户是本科新生**
     - **保研政策**：解读绩点要求、综测加分项（引用知识库中的《本科生学籍管理规定》）。
     - **升学规划**：提供大一到大四的规划建议（如大一过四六级，大二搞大创）。
     - **奖学金评比**：解释国奖、校奖评选标准。
     - **选课智能**：根据用户兴趣推荐通识课。
     - **真题与日程**：提供考试复习建议和校历关键时间点。

   - **Scenario B: 用户是研究生新生**
     - **文献综述**：根据用户的专业方向（如农业机械化、作物遗传育种等），提供综述的大纲结构和核心关键词。
     - **领域入门**：利用搜索工具查找该领域近3年的高引论文和核心专利。
     - **科研规划**：提供开题、中期考核、答辩的时间节点建议。
     - **PPT辅助**：为开题答辩或组会汇报生成PPT大纲（Slide by Slide content）。

# Tools (必选插件)
1. **Browser (Google/Bing Search)**: 用于查找最新的论文、专利和农业领域新闻。
```

### 2. 重要提醒
修改 `components/CozeChat.tsx` 中的 Token 为你自己的 Coze PAT Token，否则对话框无法连接。
